Some of my work

I combine a strong understanding of UX principles with technical and visual skills to create bold and user-friendly web apps.

  • Bear with me

    A single page JavaScript app providing rental vehicle options and information about a user-defined trip, using Mapbox GL. Aimed at the bear subculture and inspired by the rugged masculinity and colours of the '70s. This project is a WIP - I'm refactoring the code and improving the responsiveness and user flow.

  • Basque digital ordering system

    A mobile-first, responsive, hand-coded prototype of an ordering system intended for use in Basque, a tapas bar. Designed to appeal to a young adult audience with a quirky visual theme.

  • Lemonwood Vapour

    A demo WordPress ecommerce site for vaping eliquids, targeted at new and established vapers looking for a high-end, natural product.
