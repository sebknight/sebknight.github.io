Some of my work

I combine a strong understanding of UX principles with technical and visual skills to create bold and user-friendly web apps. I'm updating this all the time, so be sure to check back later. For my latest work, feel free to take a look at my GitHub!

  • Mental Health Foundation data visualisation

    This is the latest project I am working on. I designed this mobile app to present information on access to mental health services in NZ. Daniel Rhodes, Tashina Jacobs, and myself are developing the project. We're creating our own API using a Node-based server to collect and convert spreadsheet data to Google Chart-readable JSON, which then serves the data to a React front-end.

  • Bear with me

    A single page JavaScript app providing rental vehicle options and information about a user-defined trip, using Mapbox GL. Aimed at the bear subculture and inspired by the rugged masculinity and colours of the '70s. This project is a WIP - I'm refactoring the code and improving the responsiveness and user flow.

  • Basque digital ordering system

    A mobile-first, responsive, hand-coded prototype of an ordering system intended for use in Basque, a tapas bar. Designed to appeal to a young adult audience with a quirky visual theme.

  • Runaway

    A custom WordPress theme for a video production company.
